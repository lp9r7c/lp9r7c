### 🛡️ Cyber Defense Ops & Secrets Management
> "Security is not a product, but a process."

* **SIEM/XDR & Logging** : Pilotage de **Wazuh** pour l'analyse comportementale et la détection d'anomalies sur les endpoints.
* **Secrets Management** : Instance **Vaultwarden** auto-hébergée pour la gestion chiffrée des identifiants et clés API.
* **Network Hardening** : Segmentation VLAN, Tunneling **Cloudflare**, et filtrage DNS via **AdGuard Home**.
* **Offensive Security** : Pentesting régulier de l'infrastructure via **Kali Linux** et **Wireshark** pour identifier les vecteurs d'attaque.
